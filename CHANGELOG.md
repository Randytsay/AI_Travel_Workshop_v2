# Changelog - AI_Travel_Workshop_v2

All notable changes to this project will be documented in this file.

## [v2.0.0] - 2026-04-24

### Added
- **Phase 1: 靈感發想**
  - MBTI 風格 3 題人格測驗
  - 卡通頭像 Prompt（4 種風格）
  - Gemini Live 語音展示橋段

- **Phase 2: 深度規劃**
  - Gemini Deep Research 行程規劃 Prompt
  - Google Flights AI 整合教學
  - JR Pass 計算 Prompt
  - 6 種旅遊類型航班 Prompt 範本

- **Phase 3: 知識庫**
  - NotebookLM 完整教學
  - Audio Overview（行程變 Podcast）
  - 4 個實際應用場景

- **Phase 4: 視覺生成**
  - 卡通頭像生成（4 種風格）
  - LINE 分享圖 Prompt（行程圖/機票圖/總結圖）
  - Tab 切換式 UI 設計

- **Phase 5: 現場生存**
  - Google Lens 拍照翻譯教學
  - Gemini Live 即時語音翻譯
  - 4 個工作坊演示橋段
  - 飯店/餐廳/緊急狀況實用 Prompt

- **Phase 6: 商業變現**
  - Shop.com 導購文案生成
  - 多平台內容生成（IG/LINE/YouTube）
  - 5 步驟變現行動指南

- **LINE Bot 關鍵字設定**
  - 8 個關鍵字回覆模板
  - 卡通/行程/機票/翻譯/生圖完整 Prompt

### Changed
- 全面採用現代化 UI 設計（漸層 + 卡片 + 玻璃擬態）
- Mobile-first 響應式設計
- 統一配色系統（每個 Phase 有專屬主題色）
- 進度條視覺化回饋
