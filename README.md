## 📋 各 Phase 說明

| Phase | 主題 | 核心工具 | 亮點 |
|-------|------|----------|------|
| Phase 1 | 靈感發想 | Gemini 3.1 + Live | 語音輸入即時規劃，人格卡生成 |
| Phase 2 | 深度規劃 | Gemini Deep Research + Flights AI | AI 幫你算 JR Pass + 選機票 |
| Phase 3 | 知識庫 | NotebookLM | 行程表變 Podcast |
| Phase 4 | 視覺生成 | Gemini nanobanana | LINE 分享圖 + 卡通頭像 |
| Phase 5 | 現場生存 | Google Lens + Gemini Live | 即時翻譯 + 語音求助 |
| Phase 6 | 商業變現 | Gemini + Shop.com | AI 生成導購文案 |

| 關鍵字 | 功能 |
