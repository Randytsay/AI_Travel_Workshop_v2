# 貢獻指南

感謝你有興趣貢獻於 AI 旅遊工作坊 v2！

## 如何貢獻

### 1. 回報問題
發現 Bug 或有功能建議？請開 GitHub Issue，並附上：
- 使用的瀏覽器 / 作業系統
- 問題的詳細描述
- 截圖（如有）

### 2. 提交修改
1. Fork 此專案
2. 建立新分支：`git checkout -b feature/你的功能名`
3. 進行修改
4. 提交：`git commit -m 'Add: 你的功能描述'`
5. 推送到你的 Fork
6. 開啟 Pull Request

### 3. 新增 Prompt 模板
歡迎新增各國家的 Prompt 模板！
- 在 `assets/prompts/` 下建立新資料夾
- 命名格式：`[國家]-[類型].json`
- 參考現有的 `japan/` 結構

## 程式碼規範

- HTML/CSS 使用雙引號
- 變數命名使用 camelCase
- 註解使用中文
- CSS 變數統一放在 `:root` 區塊

## 問題與討論

有任何問題，歡迎在 GitHub Issues 提問！
